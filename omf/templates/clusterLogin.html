Welcome!

The Open Modeling Framework is a platform that simulates power systems behavior with an emphasis on cost-benefit analysis of emerging technologies. The platform includes programs that model distributed generation, storage, and networked controls.

Our documentation is available here, discussions and questions can be posted here. If you are interested in seeing our code, or would like to download the framework locally, the full code base along with installation instructions can be found here.

Since 2012, OMF.coop has worked with both public and private partners across the industry to transform cutting-edge research into applications that can be used by utilities and grid engineers. Our partners include The Department of Energy, Carnegie Mellon University, and a number of national labs.

Models

DISTRIBUTION ANALYSIS

Calculate the expected costs and benefits for implementing conservation voltage reduction on a given feeder circuit or determine the most economical combination of solar, wind, and storage technologies to use in a microgrid.

TRANSMISSION ANALYSIS

Calculate the expected powerflow voltage results per bus on a given transmission circuit.

RESILIENT DESIGN

Illustrate fault data, test the integrity of a feeder, and calculate outage costs given appropriate input data.

LOAD ANALYSIS

Leverage machine learning to predict load, view the financial effects of a direct load control system, or simulate the impact of demand charges in the absence of empirical data.

ENERGY STORAGE

Test the value of potential revenue streams when leveraging storage. We test for arbitrage opportunities, equipment deferral, and peak shaving.

SOLAR

Simulate the effects of distributed generation on an input grid, easily use the SUNDA Solar Costing Financing Screening Tool with a friendly interface, or run the NREL pvWatts tool for a quick estimation of solar panel output.
